# Python Automation Tool
Este proyecto automatiza la consulta de datos desde una API externa (OMDb) 
y procesa múltiples entradas mediante un sistema de lotes (Batch Processing).

## Tecnologías
* Python 3.x
* Requests library
* Git para control de versiones

## Cómo usar
1. Clonar el repositorio.
2. Crear un entorno virtual: `python3 -m venv .venv`.
3. Instalar dependencias: `pip install -r requirements.txt`.
4. Ejecutar: `python3 batch_processor.py`.
